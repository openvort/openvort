# 日报管理

当用户要求写日报、生成日报、汇总今日工作时，按以下流程操作：

## 生成流程

1. 调用 `zentao.my_tasks` 获取用户今日完成和进行中的任务
2. 调用 `zentao.my_bugs` 获取用户今日处理的 Bug
3. 按项目分组汇总工作内容

## 日报格式

```
【日报】YYYY-MM-DD 姓名

一、今日完成
- [项目A] 任务描述（耗时 Xh）
- [项目B] 修复 Bug #123：问题描述

二、进行中
- [项目A] 任务描述（进度 XX%）

三、明日计划
- 根据当前进行中的任务推断

四、问题与风险
- 如有阻塞或风险，列出
```

## 注意事项

- 使用中文名称，不用工号或英文名
- 耗时从禅道工时日志中获取，没有则不写
- 明日计划根据进行中任务合理推断，不要编造
- 如果用户没有指定日期，默认为今天
