# TigShop 开源商城系统

## 产品概述

TigShop 是江西佰商科技有限公司旗下的 100% 开源电商平台，由原 LYECS 创始人杨老师（老杨）带队开发。TigShop 致力于打造电商领域真正的 100% 开源商城系统，所有产品代码完全开源，无任何加密，支持免费商用和二次开发。

官网：https://www.tigshop.com

## 技术架构

TigShop 采用 PHP 和 Java 双版本架构，可按需选择：

- **PHP 版**：基于 ThinkPHP 8 框架
- **Java 版**：基于 Spring Boot 3 框架
- **前端**：Vue 3 + TypeScript + Uniapp + Nuxt 3
- **多端支持**：H5、微信小程序、公众号、iOS、Android、鸿蒙

系统具备高性能高并发能力，极易二次开发，并提供完善的开发文档、API 接口文档和安装部署教程。

## 产品线

### 1. 单商户标准版（永久免费）
标准的在线商品支付和管理业务流程，适合线上零售模式。支持 PC / 移动端页面装修、主题颜色一键切换、秒杀、优惠券、积分等营销功能。

### 2. 单 / 多商户 Pro 版（B2C / B2B2C）
聚焦多商户运营场景，类似京东、淘宝平台入驻模式。支持多商户入驻、各自管理店铺商品与订单，同时平台可统一管控与数据汇总。

### 3. B2B2B 企业交易 / 批发平台 Pro 版
面向企业或批发用户，类似阿里巴巴模式。支持会员企业认证、登录查看价格、多规格批量下单、询价、多商户多店铺，并内置 IM 客服系统。

### 4. S2B2C 供应商系统
供应商可发布和管理商品，平台 / 店铺可直接上架供应商商品，设置价格浮动比例或金额，还可一键导入供应商商品实现独立铺货。支持 S2B2C / S2B2B / S2B2B2C / S2B2B2B 模式。

### 5. O2O 多门店线上线下一体化系统
支持管理店铺入驻，到店服务、服务预约、订单核销等功能模块，并支持 POS 收银台、到店自提。适用于商圈、本地化生活服务平台及垂直电商。

### 6. 跨境多语言 Pro 版
基于 Vue3、TypeScript、Uniapp、i18n 国际化组件架构，前台支持数十种语言自由切换，同时支持多种货币切换。支持跨境支付接口和跨境物流接口对接。

### 7. 分销系统
完善的分销体系，提供分销员管理、佣金、结算等核心功能。支持多级分销模式和灵活佣金方案配置，配备分销数据中心、内容管理工具、分销海报生成等。

### 8. TigChat 在线 IM 客服系统
为商城提供完整的在线即时通讯客服能力。

## 支持的运营模式

| 模式 | 说明 |
|------|------|
| B2C | 标准零售，单品牌直销 |
| B2B2C | 多商户入驻，平台 + 商家模式 |
| S2B2C | 供应链模式，供应商 + 平台 + 消费者 |
| O2O | 线上线下融合，多门店 + 到店服务 |
| B2B | 企业对企业批发交易 |
| 跨境电商 | 多语言 + 多货币 + 跨境支付物流 |

## 核心功能

- **页面装修**：PC 端和移动端均支持可视化页面装修
- **主题切换**：主题颜色一键切换
- **营销工具**：秒杀、优惠券、积分、满减等
- **订单管理**：完整的订单生命周期管理
- **支付对接**：微信支付、支付宝等主流支付
- **物流追踪**：物流信息查询与管理
- **数据统计**：运营数据报表与分析
- **权限管理**：多角色权限体系
- **IM 客服**：内置在线客服系统
